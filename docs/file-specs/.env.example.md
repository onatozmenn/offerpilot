# File Spec: `.env.example`

## Status

`specified`

## Purpose

Document all runtime environment variables with safe local placeholders.

## Depends On

- `docs/07-security-privacy.md`
- `internal/config/config.go` contract.

## Public Surface

- `OFFERPILOT_HTTP_ADDR=127.0.0.1:8080`
- `OFFERPILOT_DATABASE_URL` with a placeholder local PostgreSQL URL.
- `OFFERPILOT_DATABASE_MAX_CONNS=10`, `OFFERPILOT_DATABASE_MIN_CONNS=1`, `OFFERPILOT_DATABASE_MAX_CONN_LIFETIME=30m`, `OFFERPILOT_DATABASE_MAX_CONN_IDLE_TIME=5m`, and `OFFERPILOT_DATABASE_HEALTH_CHECK_PERIOD=30s`.
- `OFFERPILOT_LOG_LEVEL=info` and `OFFERPILOT_LOG_FORMAT=text`.
- `OFFERPILOT_CORS_ALLOWED_ORIGINS=http://localhost:5173` as a comma-separated exact-origin list.
- `OFFERPILOT_HTTP_READ_HEADER_TIMEOUT=5s`, `OFFERPILOT_HTTP_READ_TIMEOUT=10s`, `OFFERPILOT_HTTP_WRITE_TIMEOUT=30s`, `OFFERPILOT_HTTP_IDLE_TIMEOUT=60s`, and `OFFERPILOT_HTTP_MAX_BODY_BYTES=1048576`.
- `OFFERPILOT_SHUTDOWN_TIMEOUT=15s` and `OFFERPILOT_OUTCOME_MAX_FUTURE_SKEW=2m`.
- `OFFERPILOT_SIM_MAX_REQUESTS_PER_SECOND=100`, `OFFERPILOT_SIM_MAX_DECISIONS=100000`, `OFFERPILOT_SIM_MAX_WORKERS=32`, `OFFERPILOT_SIM_MAX_DURATION=30m`, and `OFFERPILOT_SIM_MAX_ERRORS=100`.
- `OFFERPILOT_API_PROXY_TARGET=http://127.0.0.1:8080` for Vite development only.
- Empty `VITE_API_BASE_URL` to select the browser's same origin.

## Required Behavior

- Contain no real credentials.
- Use the Compose service hostname only in a clearly labeled container value; provide a localhost alternative in comments.
- Include the exact names/defaults above and explain that Compose overrides hostnames where required.
- Match variable names parsed by Go config and consumed by Vite.

## Failure Cases

- A required variable is undocumented or a placeholder resembles a real secret.

## Non-Goals

- Loading environment files in application code.

## Validation

- Compare keys against config tests and `docker compose config` output.

## Completion Checklist

- [ ] Every supported variable appears once.
- [ ] Values are safe placeholders.
- [ ] Focused validation passes.
